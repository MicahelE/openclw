## Productivity

### Cognitive Memory

**Rating: Safe** | by icemilo414

A persistent memory layer that lets AI agents recall past interactions and build on previous context. Only requires file system access — no network, no shell execution. Essential for long-running projects.

```bash
claw install icemilo414/cognitive-memory
```

### Self Improving Agent

**Rating: Safe** | by pskoett

Captures learnings, errors, and corrections to enable continuous improvement. Your agent gets smarter over time by tracking what works and what fails. The most popular skill on ClawHub for good reason.

```bash
claw install pskoett/self-improving-agent
```

### Pndr

**Rating: Safe** | by dgershman

Intelligent note-taking that captures thoughts, generates summaries, and surfaces relevant past notes based on current context. Great for building a personal knowledge base.

```bash
claw install dgershman/pndr
```
