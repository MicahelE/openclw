## Productivity

### Smart File Organizer

**Rating: Safe** | by ClawLabs

Automatically sorts files into folders based on content type, date, and custom rules. We love this one because it only requests file system access — no network, no shell execution.

```bash
claw install smart-file-organizer
```

### Email Summarizer

**Rating: Caution** | by InboxZero

Connects to your inbox and generates daily digests with priority sorting. The caution rating is because it requires email account access via OAuth — a significant permission, but handled properly through standard OAuth flows.

```bash
claw install email-summarizer
```

### Meeting Notes AI

**Rating: Caution** | by ProductivityAI

Transcribes and summarizes meetings with action item extraction. Requires microphone access and meeting platform integration. Great for teams but review the permissions carefully.

```bash
claw install meeting-notes-ai
```
